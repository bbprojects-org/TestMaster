  EDITING A QUESTION SET

Question 0 - Question 0 is not actually part of the question set but is used to enter the summary information seen on the initial TestMaster screen. If the first optional answer is checked then the order of all questions in the set will be randomised when the test is run, if unchecked the question order will be as entered. If a graphic is provided as an exhibit it will be used as the logo above the summary information in the initial TestMaster screen, and in the run test window.
Note: If a logo is provided, the colour of the bar in the run test screen is set to that of the top left pixel of the logo for visual continuity. Maximum height of this graphic is 120 pixels.

At the left side of the screen:

  • Question - the text of the question.
  • Possible answers:
    • Randomise - if this check box is ticked then the order of the answers will be randomised. If not ticked then the order of the answers will be as entered.
    • Optional answers text is entered in the boxes provided (maximum of eight options).
    • The buttons to the left of each answer select the correct answer(s).
  • Exhibit - a graphic may be included for each question. The easiest way to add/change an image is to drag and drop it on the window. If a question has an associated exhibit it will automatically be shown, and the Add button will be replaced by Change. The Show button, enabled when an Exhibit is attached, displays the graphic full size in its own window.
  • Reference / Comment - (optional) this is the text shown with the answer when the Answer button is pressed or when incorrect questions are reviewed. The text should explain the answer and any relevant reference(s).

At the right of the screen:

  • Question #N - the current question being edited.
  • Add - adds a question to the set. Puts the system in edit mode (only Save/Cancel buttons enabled)
  • Delete - deletes displayed question, subject to confirmation.
    Note: Question 0 (summary) cannot be deleted
  • Save - saves current changes in the question set, and restores View mode
  • Cancel - cancel any changes to this question, and puts display in View mode
  • Navigation buttons - First : Previous : Next : Last. Allow the user to move through the question set.
  • Find - used to search for a phrase within the question set. A dialogue box is displayed where the search text can be entered and the First/Next occurrence searched for. This dialogue box and the edit window can be used at the same time.
  • Answers: Single / Multiple - Selects the number of permitted responses (Single displays radio buttons as the options, Multiple displays check boxes).
  • Close - stop editing, and return to the main TestMaster screen.
