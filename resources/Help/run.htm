  RUNNING A TEST

Each question in the set is displayed in turn, the order being randomised if this option has been selected by the question set author, with the question text appearing first followed by the list of possible answers. If an exhibit (graphic) has been included for the question this will also be displayed.

Answers may be selected by clicking on the radio button / check box or the answer text itself.

At the bottom of the screen:

  • Navigation buttons:
    • Previous - moves to the Previous question in the set
    • Next - moves to the Next question in the set
  • Answer - if enabled, this shows the answer to the current question and any comments / explanation provided.
    Note: This option may be disabled via the Administrator option.
  • End Test - stops the timer, evaluates the results and enables the review button if errors were made.
    Note: TestMaster keeps a tally of all the questions viewed (whatever order they are viewed in) and evaluates the result based on the number of questions seen and how many of those were answered correctly. This allows the user to get a fair representation of their success even if only part of the test is completed
  • Review - the 'End Test' button when pressed becomes the Review button if any answers were incorrect. All incorrect answers can then be reviewed using the Next/Previous buttons. The correct answer and comments (if provided) will be displayed along with the user's incorrect answer.
  • Save Results - on completion of a test the evaluated results can be saved to the summary file. If an attempt to close the window is made without saving the results the user is prompted to save them.
  • Close - Close this window.

At the top of the screen:

  • Mark - sets the current question as marked (or clears the mark) so that it may be reviewed later. To review marked questions use the Goto button which shows all marked questions.
  • Goto - opens a side panel with a list of question numbers to permit the user to select/view a question by clicking on the number. This is particularly useful to go to Marked questions highlighted with '**'. Close this window by clicking on the Goto button again.
