  INSTALLATION

TestMaster on Mac

Unzip the program zip file and copy TestMaster.app to your Applications folder. The help files and other resource files are automatically copied in the TestMaster.app bundle.

TestMaster on Windows

The best way to install TestMaster is to unzip the program zip file using the 'Use folder names' option in WinZip. This will produce the following folder structure:

\Program Files\TestMaster\

TestMaster.exe
logo.png

\Program Files\TestMaster\Help

These help files

Create your own shortcut to the TestMaster.exe file to complete the installation.

Question Sets

Question sets (identified by the TMQZ extension) are ZIP files which include the question set database and any associated exhibits/graphics. The easiest way to install them is to drag and drop them on to the main TestMaster window. TestMaster will then copy them to the correct folder whether Windows or MacOS is being used. Alternatively copy them manually to:

Windows
\Program Files\TestMaster\Tests\

MacOS
~/Library/Application Support/TestMaster/Tests/
