  THE MAIN TESTMASTER WINDOW

The available menu items are as follows:

Application - dropdown menu as follows:

  • Results - displays the results summary window. The Clear button, if enabled via the Administrator option, can be used to erase the contents of the results file.
  • Show Hints - shows (or not) hints when the pointer is moved over window objects, buttons etc.
  • Font Size - sets the font size to Small, Medium or Large depending on user's preference.
  • Elapsed Time - sets test time to run from zero hence indicating elapsed time since start of the test.
  • Time Remaining - sets time to countdown from a set time. A dialogue box appears when the test is started requesting a time limit (default 60 minutes). TestMaster remembers the last time limit set.
  • Administrator - opens Administrator dialogue box (password protected).

Help - dropdown menu as follows:

  • TestMaster Help - launches the system default web browser to display help information.

The left listbox displays all the available tests in the Tests directory (identified by the .tmqz file extension). The right section displays the summary information for the selected file, if any has been provided.

The six buttons along the bottom are used as follows:

  • Start - run the currently selected test.
  • Edit - edit the currently selected question set.
  • Add - add a new question set. Asks for a name for the new question set, creates an empty file then puts TestMaster in edit mode.
  • Delete - delete the selected question set.
  • Help - show these help pages (uses the default web browser on the computer).
  • Exit - exit the program.
    • Note: The Add/Edit/Delete buttons may be disabled via the Administrator option.
