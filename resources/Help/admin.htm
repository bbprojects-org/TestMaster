ADMINISTRATOR

This dialogue window is reached via the Application | Administrator menu option, unless read-only mode has been selected in which case the option does not appear but access can still be achieved via Command-A on Mac (Ctrl-A on Windows). This facility is password protected.

There are various options:

  • Enable answers button - the Answer button appears on the run test screen and allows the user to 'preview' the answer to the question. Remove the check in this box to disable this facility.
  • Enable clear results button - the Clear button appears on the Results Summary window. Remove the check in this box if the user should not be able to clear the results file.
  • Enable running score - when checked the current score is displayed and updated throughout the test. If not checked it is only displayed when the End Test button is pressed or, if in timed mode, the permitted time expires.
  • Enable question sets Add/Edit buttons - removing the check in this box sets read-only mode and the user cannot edit, add or delete question sets. Setting this mode also hides the Administrator option on the Application menu.
  • Enable question sets Delete button - if not using read-only mode but you still want to prevent question sets being deleted, then remove the check in this box.
  • Set test screen colour - allows the background colour of the test screen to be changed.
  • Change Password - when first installed the Administrator password is set to fibonacci. Adminstrators should change this!
    Note: There is no request for the original password first since this had to be entered to reach the Administrator window. Do not leave the computer unattended with the Administrator dialogue open if there is a possibility someone could change the password!
  • Reset to defaults - restores default options.

OK - pressing the OK button implements any changes that have been made.

CANCEL - pressing the Cancel button exits the window without making any changes to the options.
